Very Simple Instructions

1. Push the start button

2. Press the blue okay (or close) button on the pop-up window

3. Find the number to your name i.e. "1) Noah" would be 1

4. Go down to the text box that says "Enter your number" and put in your number

5. Push the button that says "Find Giftee" and BOOM you're done
